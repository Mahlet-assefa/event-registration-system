# Event Registration System

A full-stack event registration system built with React frontend and Express backend.

## Features
- User authentication (Login/Register)
- Event browsing and registration
- User dashboard
- Admin event management

## Tech Stack
- **Frontend**: React 18, Bootstrap 5
- **Backend**: Node.js, Express.js
- **Database**: JSON files (simulated database)

## Project Structureevent-registration-system/
├── client/ # React frontend
├── server/ # Express backend
├── docs/ # Documentation
├── tests/ # Test suites
├── releases/ # Release packages
└── scripts/ # Build/deploy scripts

## Getting Started
1. Clone the repository
2. Run `npm install` in both client and server directories
3. Start development servers:
   ```bash
   cd client && npm start
   cd server && npm start
