# Release Notes - v1.0

**Release Date:** 2025-12-24
**Baseline:** BL1

## Initial Working System
This release establishes the core foundation of the Event Registration System.

### Features
- **Project Structure**: Standardized directory layout (`src`, `docs`, `data`, `tests`).
- **Backend Scaffold**: Express server setup with initial logging middleware.
- **Frontend Scaffold**: React + Vite application with basic styling.
- **Documentation**: Initial API endpoints guide and setup instructions.

### Known Issues
- Basic functionality only; authentication and event management to be expanded in v1.1.
